# RisingHorn Group Website (Next.js + FastAPI on Vercel)

## Run locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## API
- GET /api/health
- POST /api/contact

## Logo
Replace `public/logo.svg` with your finalized logo (keep the same filename),
or update the reference in `components/Navbar.tsx`.
